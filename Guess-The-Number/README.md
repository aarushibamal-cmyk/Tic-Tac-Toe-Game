# 🎯 Guess The Number

A fun browser-based number guessing game built with vanilla HTML, CSS, and JavaScript.

## 🎮 How to Play

1. A random number between **1 and 100** is generated
2. Enter your guess in the input field and click **Submit Guess**
3. You get feedback — *Too High* or *Too Low* — after each guess
4. You have **10 attempts** to guess correctly
5. Click **New Game** to reset and play again

## 🚀 Getting Started

No installs or dependencies needed. Just open `index.html` in your browser.

```bash
git clone https://github.com/aarushibamal-cmyk/guess-the-number.git
cd guess-the-number
open index.html
```

## ✨ Features

- Random number generated on every new game
- Live feedback on each guess (Too High / Too Low)
- Attempts counter counts down in real time
- Previous guesses displayed on screen
- Inline validation messages (no browser alerts)
- New Game button resets everything cleanly
- Fully responsive, no frameworks

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, gradients, transitions)
- Vanilla JavaScript (DOM manipulation, AbortController)
