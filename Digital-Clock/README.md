# ⏰ Digital Clock

A simple and responsive Digital Clock built using HTML, CSS, and JavaScript. The clock displays the current hours, minutes, and seconds in real time with a effect whenever the time changes.

## 🚀 Features

- Real-time clock updates
- Displays Hours, Minutes, and Seconds
- Flip animation on time change
- Responsive design
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)


## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/aarushibamal-cmyk/digital-clock.git
```

2. Navigate to the project folder:

```bash
cd digital-clock
```

3. Open `index.html` in your browser.
