# JavaScript, HTML & CSS Learning

This repo tracks my journey learning JavaScript — from DOM basics to async/await — through hands-on mini projects, along with my personal notes on core JS concepts.

Following along with the **Chai aur Code** JavaScript playlist by Hitesh Choudhary, plus independent practice.

## 📁 Projects

| Project | Description |
|---|---|
| [`BMI---Calculator`](./BMI---Calculator) | BMI calculator built with vanilla JS — takes height/weight input and calculates BMI category |
| [`Color-Changer`](./Color-Changer) | Simple JS project using event listeners to dynamically change background colors |
| [`Digital-Clock`](./Digital-Clock) | Real-time digital clock built using JS `Date` object and DOM updates |
| [`Guess-The-Number`](./Guess-The-Number) | Number guessing game — practiced conditionals, loops, and DOM interaction |
| [`amazon_clone`](./amazon_clone) | Clone of Amazon's product listing UI with a built-in to-do list feature using JS, HTML, and CSS |
| [`expense_tracker`](./expense_tracker) | Expense tracker app using JS, HTML, CSS — DOM manipulation and localStorage for persisting data |
| [`tic-tac-toe`](./tic-tac-toe) | Classic Tic Tac Toe game — logic for turns, win conditions, and board state |
| [`to-do-app`](./to-do-app) | To-do list app — practiced Promises, async, and await |
| [`weather_app`](./weather_app) | Weather app using the OpenWeather API — handled fetch calls, async data, and error handling |

## 📝 Notes

Alongside the projects, this repo includes my personal notes covering:
- DOM manipulation & event listeners
- `localStorage` and browser storage
- Async JS — callbacks, Promises, `async`/`await`, `fetch`
- Array methods (`map`, `filter`, `reduce`, etc.)
- Core JS fundamentals (objects, functions, scope, closures)

## 🛠️ Tech Stack

This repo is a stepping stone before moving into React and the rest of the full-stack path (Node.js, Express, MongoDB). Each project was built to reinforce one or more core JS concepts before relying on a framework.
